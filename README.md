# A Tela de Cadastro da Plataforma Dio com React

#### Projeto desenvolvido com React

### Tecnologias
- Javascript

### Como rodar o projeto

1 - Clone o repositório

2 - Instale as dependeências
    
    npm install

3 - Execute o projeto

    npm start
